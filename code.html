PacBio Software

View, download, and contribute bleeding edge code for production and research. Access and/or use is subject to the DevNet Terms of Use.



Officially supported software

    Exclusive and escalated support via the Customer Portal where enquiries will be answered within 2-3 days. Enquiries on GitHub issues will be answered at developer's discretion.

  • SMRT® Analysis: Powerful software suite for sequence alignment, assembly, variant detection, and base modification discovery.
 Developer supported software

    Developer support via GitHub issues. Enquiries will be answered at developer's discretion

Utilities

  • Blasr: C++ long-read aligner for PacBio reads (Basic Local Alignment with Successive Refinement).
  • pbcore: Python module for processing PacBio data files and building PacBio bioinformatics applications.
  • pbh5tools: Python toolset (depends on pbcore) for interrogating PacBio® HDF5 files (cmp.h5, bas.h5).
  • R-pbh5: R package for interrogating PacBio HDF5 files (cmp.h5, bas.h5).
  • R-pbutils: R package with various plotting and analysis utilities.
  • stsplots: R script to plot primary analysis quality control metrics.

Variant Detection

  • ConsensusCore: C++ library of multiple-sequence consensus routines with SWIG bindings.
  • Genomic Consensus: Python script for variant and consensus calling (Quiver algorithm).
  • Correlated Variants: Python script that quantifies relationships between co-located variants.

Genome Assembly

  • HGAP-script: Shell script to run the Hierarchical Genome Assembly Process with advanced tuning parameters (depends on SMRT Analysis.

Base Modification

  • R-kinetics: R scripts for detecting DNA modifications from single molecule, real-time sequencing data.
  • Kinetics Tools: Python scripts for detecting DNA modifications from single molecule, real-time sequencing data.
  • Motif Maker: Scala wrapped java scripts for finding methylated motifs in prokaryotic genomes.
  • Motif Finder: GUI Windows program that will analyze your FASTA file and the modifications.gff file.
  • Motif-Finding: R scripts for kinetic data QC and motif finding.

SMRT View Genome Browser

  • SMRT View: Stand-alone genome browser for PacBio data.
