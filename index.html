Pacific Biosciences DevNet

DevNet supports analysis of sequence data produced from the Pacific Biosciences DNA sequencing platform. Open source software, data, documentation and tips for PacBio sequencing can be found here.

News and Features

Introducing the new P5 polymerase and C3 chemistry combination (P5-C3), which results in industry-leading N50 read lengths of 10,000 base pairs.

SMRT Analysis 2.1 is a pre-requisite for analyzing data from the PacBio RS II and the P5 enzyme. Customers: please contact your local Field Account Specialist in order to upgrade and take advantage of the following features:

  • 10-20x faster microbial genome assembly using HGAP 2.0 in SMRT Portal
  • New Reads of Insert protocol to estimate of the length of the insert sequence loaded onto a SMRT® Cell. Reads of Insert replaces the Circular Consensus Sequencing (CCS) protocol, which has been moved off the primary analysis instrument.
  • Long amplicon analysis (beta) to determine phased consensus sequences
  • BridgeMapper visualization tool (beta) for alignment and assembly analysis

Watch the HGAP genome assembly webinar (Best on Chrome) for an overview.


Data Release Announcement:

The dataset for the 10x long-read, shotgun coverage of a human genome sample (hydatidiform mole) that was sequenced on the PacBio® RS II platform is available for download. For more information, please see the PacBio Blog entry.
